# Legacy Public Archive

This directory contains prior full-site component markup, scroll-enhancement
JavaScript, and social-card assets kept for reference while the public site is a
temporary placeholder.

The active site does not import this directory. `tsconfig.json` excludes it from
active Astro and TypeScript checks, and these files should not be moved into
active source or `public/` unless the full-site rewrite intentionally needs
them.
